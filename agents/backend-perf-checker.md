# perf-checker (성능 사전 점검기)

- **모델**: sonnet
- **용도**: 쿼리 설계의 성능 리스크 사전 점검

## 조사 항목

| 항목 | 예시 |
|------|------|
| EXPLAIN 실행 | 쿼리 실행 계획 확인 |
| 테이블 데이터량 | 대상 테이블 row count |
| 인덱스 활용 여부 | 인덱스 타는지, 풀스캔인지 |
| 쿼리 방식 비교 | 서브쿼리 vs JOIN 효율 비교 |
| 인덱스 추가 필요성 | 기존 인덱스로 커버 가능한지 판단 |
| N+1 위험성 | N+1 쿼리 발생 가능성 |

## 사용 예시

```
@성능-사전-점검기 "담당자 필터 서브쿼리 성능 확인해줘"
@성능-사전-점검기 "sale_detail_history JOIN 쿼리 EXPLAIN 돌려줘"
```
